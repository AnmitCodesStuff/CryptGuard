
# CryptGuard

A powerful encryption tool that keeps your sensitive information safe. Encrypt and decrypt text instantly with custom keys.

## Run Locally

Clone the project

```bash
  git clone https://github.com/AnmitCodesStuff/CryptGuard.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Documentation

- Use Encrypt Mode (Key -> Msg = Code)

- Use Decrypt Mode (Key -> Code = Msg)
